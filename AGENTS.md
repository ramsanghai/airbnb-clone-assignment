# AI Agents

This project used AI-assisted development for implementation support, UI refinement, debugging and code review.

## UI Review Agent
- Compare layout with the reference
- Check spacing and alignment
- Check typography and image proportions
- Review hover and interaction behaviour

## Code Quality Agent
- Keep React components organised
- Review TypeScript issues
- Avoid unnecessary duplication
- Keep code readable and maintainable

## Accessibility Agent
- Check aria labels
- Check keyboard navigation
- Review dialog semantics
- Check lightbox arrow navigation and close behaviour
