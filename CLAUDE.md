# Claude Development Instructions

## Project
Airbnb listing page clone built with React, TypeScript and Vite.

## Development Guidelines
- Keep components small and readable
- Use TypeScript
- Reuse shared listing and photo data
- Match the reference UI closely
- Preserve accessibility labels
- Avoid unnecessary dependencies
- Do not change unrelated components when fixing an issue

## UI Priorities
- Visual fidelity
- Correct spacing
- Image proportions
- Photo Tour behaviour
- Lightbox navigation
- Sticky section behaviour
- Keyboard accessibility

## Validation
Before completion:
- Run the production build
- Check TypeScript errors
- Test all image interactions
- Test keyboard navigation
- Compare the final page with the reference
