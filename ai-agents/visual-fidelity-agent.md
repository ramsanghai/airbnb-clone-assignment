# Visual Fidelity Review

## Goal
Compare the implementation with the supplied reference at the same desktop viewport.

## Review
- Overall container width
- Horizontal and vertical spacing
- Font size, weight and line height
- Border radius
- Border and divider colors
- Image crop and aspect ratio
- Hover states
- Sticky behavior
- Modal sizing
- Lightbox spacing
- Photo Tour layout

## Output
Return only concrete differences and the smallest code change needed for each difference.
